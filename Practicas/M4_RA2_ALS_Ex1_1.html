1. Elementos de texto

Comparación de <em> y <i>, <strong> y <b>:

El elemento <em> se usa para resaltar un texto con énfasis, mientras que el elemento <i> solo representa texto en cursiva sin implicar énfasis. Por ejemplo: Este es un texto con énfasis vs. Este es un texto en cursiva.

Por otro lado, el elemento <strong> se usa para denotar un texto de fuerte importancia, mientras que <b> solo pone el texto en negrita sin agregar importancia semántica. Ejemplo: Texto importante vs. Texto en negrita.

Uso de <q> para Citas Cortas:

En el siguiente ejemplo utilizamos el elemento <q> para una cita breve dentro del texto: La simplicidad es la máxima sofisticación dijo Leonardo Da Vinci.

Crear Subíndices y Superíndices:

Subínices: La fórmula química del agua es H2O.

Superíndices: La fórmula de la velocidad en física es v = d/t2.

Uso de <pre> para Texto Preformateado:

El siguiente ejemplo muestra cómo usar el elemento <pre> para texto que requiere un formato específico: 

                    Este es un texto preformateado
                    donde los espacios y los saltos
                    de línea se respetan.
